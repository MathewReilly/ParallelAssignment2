# ParallelAssignment2

To run with gcc:
  gcc -lstdc++ -std=c++11 -o cupcake.out Assignment2P1.cpp

  ./cupcake.out

  gcc -lstdc++ -std=c++11 -o showroom.out Assignment2P2.cpp

  ./showroom.out

To run with g++:
  g++ -std=c++11 -o cupcake.out Assignment2P1.cpp

  ./cupcake.out

  g++ -std=c++11 -o showroom.out Assignment2P2.cpp

  ./showroom.out
